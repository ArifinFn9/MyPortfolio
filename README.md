# Muhammad Arifin — Personal Portfolio

A premium, modern, and highly responsive bilingual (English / Indonesian) personal portfolio website. Built using Next.js 16 (App Router), next-intl, Framer Motion, and Tailwind CSS, this portfolio showcases expertise in finance, data entry, and administrative systems.

## ✨ Core Competencies & Focus
- **Financial Data Analysis**: Multi-year corporate statement analysis, financial ratio modeling, cash flow visualizations.
- **Data Entry & Management**: Processing databases with high precision, transaction auditing, database cleaning.
- **Office Administration**: Secure documentation, verified banking records, administrative workflow optimization.

## 🚀 Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **Next.js 16 (React 19)** | High-performance React framework with App Router |
| **next-intl** | Seamless bilingual routing and localization (EN / ID) |
| **Tailwind CSS 4** | Box-shadow, backdrop blur, and custom utility classes |
| **Framer Motion 12** | Smooth layout animations and physics-based interactions |
| **Lucide React** | Consistent and premium iconography |

## 📁 Key Project Structure
- `app/` - Page routes and global configurations.
- `components/` - Reusable UI widgets and sections (Hero, Navbar, Footer, Projects, Skills).
- `data/` - Centralized data assets (projects database, social links, resume URL).
- `messages/` - Localization JSON files (`en.json` and `id.json`) for instant bilingual updates.
- `public/assets/` - Static assets including images, certificates, and documents.

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- npm (Node Package Manager)

### Installation & Local Run
```bash
# 1. Install dependencies
npm install

# 2. Start the local development server
npm run dev

# 3. Build the optimized production bundle
npm run build

# 4. Start the production server
npm run start
```
Once started, open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.

## 🌐 Centralized Configurations
The portfolio is designed to be easily editable without touching JSX files:
- **Social Media & Credentials**: Edit [data/socials.js](file:///C:/Users/arifi/Desktop/MyPortofolio/data/socials.js) to update handles, links, and PDF certificate files.
- **Projects Database**: Edit [data/projects.js](file:///C:/Users/arifi/Desktop/MyPortofolio/data/projects.js) to add or manage your projects, tech stack badges, and image cards.
- **Bilingual Copywriting**: Edit translation files inside the [messages/](file:///C:/Users/arifi/Desktop/MyPortofolio/messages) folder to update biography texts, project descriptions, and headings.
